# I.T. Engineering Final Project 
 ***islandMovers Taxi Company***
## Overview
I.T. Engineering Final Project is a fullstack web application created as part of my group assignment in COMP 2415 Information Technology Engineering at The University of the West Indies Cave Hill Campus in 2024. This project serves to provide 6 taxi applications for a St. Vincent taxi company called islandMovers, with businesses operating in Barbados. Dominca, Florida, Dubai and Lebanon.

As the assigned software engineer, I was tasked with creating 6 applications using 6 databases (created by another member of the group), requiring the incorporation of my ability to build and design a software which meets the needs of the company, including database integration and SQL injection prevention.
## Features
	- Login system
		Only for customers to manage their ordered rides
	- Ride Order System
		Users can order a ride and see their pending rides
	- Location Detection
		The software detects the user's country to ensure the correct application is being used
		Restricts usage if user is not using the correct application
## Technologies Used
